📊 Onglet Calcul
Mode Simple — 1 clic = 1 jour au TJ du coefficient actif.
Mode Détaillé — Saisie heures + coefficient par jour (utile si le poste change en cours de mission).

⏳ — Met la mission en attente sur ce mois précis. Bloc bleu visible uniquement sur le bon mois.
✓ Fin — Archive le snapshot. Choix : garder ou remettre à zéro les jours pour la mission suivante.
⚖ Comparatif — Apparaît en bas si une fiche du mois est disponible. Affiche l'écart pointeuse vs fiche.
⚙️ Onglet Params — Coefficients
A0 / B0 / C0 / C1 / C2 / C5 — un clic met à jour TJ, taux horaires, primes et HSE (frais pro) automatiquement.

TJ calibrés : A0=142€ · B0=149€ · C0=177€ · C1=188€ · C2=199€ · C5=229€

IFM (10%) + CP (10%) inclus dans le TJ. Base cotisable = TJ ÷ 1,20.
Prime assiduité (503) s'ajoute au net mensuel, pas au net/jour.

Timeline — Historique multi-années. Boutons 📦 Archivées (avec 🗑 Supprimer) et ⏳ En attente (avec ▶ Activer).

⚖ Flux comparatif (recommandé)
1. Pointe le mois complet
2. ✓ Fin mission → archive
3. Reçois ton bulletin → upload dans Docs
4. Le comparatif s'ouvre : tableau pointeuse vs fiche avec écarts en €
5. Bouton "Analyse IA complète" pour vérification légale approfondie
🧮 Formule de calcul
Brut = TJ × jours (ou Σ heures × taux)
Cotis = Brut × 20,22%
Net Social = Brut − Cotis
Net avant impôt = Net Social + Repas NS + HSE
PAS = Net Social × 0,7699 × 6%
NET À PAYER = Net avant − PAS + Assiduité
❓ Mes données sont-elles sauvegardées ?
Oui, tout est stocké automatiquement sur ton appareil (localStorage). Aucune connexion requise. Les données restent tant que tu ne vides pas le cache du navigateur.
❓ Quel coefficient choisir ?
Regarde ton contrat agence. Le coefficient correspond à ton poste : C0 = Technicien Sismique standard (TJ 177,72€). Si tu changes de poste en cours de mission, tu peux changer le coefficient jour par jour en Mode Détaillé.
❓ IFM et CP sont-ils comptés deux fois ?
Non. IFM (10%) + CP (10%) sont déjà intégrés dans le TJ. La base cotisable = TJ ÷ 1,20. Ils ne s'ajoutent pas en plus.
❓ 2 missions sur le même mois ?
Pointe Mission A → ✓ Fin → "Remis à zéro" → saisis Mission B → continue à pointer. Les deux sont archivées séparément dans la Timeline. Les totaux Data/Mois les cumulent correctement.
❓ Le NET JOUR est différent du NET À PAYER — pourquoi ?
Normal. NET JOUR = ce jour seul. NET À PAYER = total mensuel qui inclut la prime assiduité (versée une seule fois par mois, pas par jour). L'explication est affichée sous le net/jour en Mode Détaillé.
❓ Comment lancer le comparatif fiche vs pointeuse ?
Flux recommandé : pointe le mois → ✓ Fin mission → archive → une fenêtre propose d'uploader ta fiche ou de lancer le comparatif si elle est déjà là. Tu peux aussi le faire depuis Docs → ouvre la fiche → "Lancer la comparaison".
❓ Le PDF multi-pages est-il analysé entièrement ?
Oui. Le PDF est envoyé directement à l'API Claude (pas de conversion image). Toutes les pages sont lues — utile quand ta fiche la fiche s'étale sur plusieurs feuillets.
❓ Comment supprimer une archive ?
Data → Timeline → active le toggle 📦 Archivées → bouton "🗑 Supprimer" sur l'archive concernée → confirmation requise.
❓ Comment exporter ma déclaration d'impôts ?
Data → Année → "📄 Exporter récap fiscal". PDF avec cases 1AJ (salaires nets imposables) et 8HV (PAS prélevé). Estimations PayTrack — vérifier avec les fiches officielles.
❓ Les paniers repas sont-ils soumis aux charges ?
Non. NS = Non Soumis. Ils s'ajoutent au net social sans cotisations ni impôt (plafond URSSAF : 20,20€/repas). La HSE (frais pro) est également NS.
✏ Personnaliser les postes
Modifie les labels et taux journaliers selon ton secteur. Les taux horaires restent configurables dans Params.
